# 🩺 Skin Disease Detection AI

An AI-powered skin disease detection system built using CNN, Flask, OpenCV, and SQLite. The project provides real-time skin disease prediction, confidence scoring, heatmap visualization, and scan history management through an interactive medical dashboard UI.

---

## 🚀 Features

- AI-based skin disease prediction
- CNN deep learning model
- Heatmap visualization
- Real-time image upload & analysis
- Modern animated dashboard UI
- Scan history tracking
- SQLite database integration
- Flask backend API
- Responsive frontend design

## 📊 AI Prediction Workflow

1. Upload skin image
2. CNN model processes image
3. Disease prediction generated
4. Confidence score displayed
5. Heatmap visualization created
6. Scan saved into database
